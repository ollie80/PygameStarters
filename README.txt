Pygame Starters By Ollie :)


Classes:
    
- For simple sprites use the Generic class

- For animated sprites use the Animated class

- For particles use Particle class

- For a timer use the Timer class

- If you want sprites to move when your player moves use Camera Group class as your main group

Functions:

- For assets used in Animated and Particles use the Import Folder function

- For importing folders as a dictionary use the Import Folder Dict function
